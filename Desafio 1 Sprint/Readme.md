# Calculadora de IMC em JavaScript

Este projeto fornece uma simples calculadora de Índice de Massa Corporal (IMC) em JavaScript. O IMC é uma medida utilizada para avaliar se uma pessoa está em um peso saudável em relação à sua altura.

## Funcionalidades

- Cálculo do IMC a partir do peso e altura fornecidos pelo usuário.
- Classificação do IMC de acordo com as faixas definidas pela Organização Mundial da Saúde (OMS).
- Interatividade simples através de prompts e alerts no navegador.
